Política de Privacidad

Última actualización: Marzo 2024

Su privacidad es importante para nosotros. Esta Política de Privacidad describe cómo recopilamos, usamos, divulgamos y protegemos su información cuando utiliza nuestra aplicación.

1. Información que recopilamos

Datos personales

Cuando se registra o utiliza nuestra aplicación, podemos recopilar:

  • Nombre
  • Dirección de correo electrónico
  • Otros detalles que proporcione voluntariamente

Datos de uso

Recopilamos automáticamente:

  • Tipo de dispositivo y sistema operativo
  • Versión de la aplicación
  • Patrones de uso e interacciones
  • Marcas de tiempo de acceso

2. Cómo utilizamos su información

  • Para proporcionar y mantener nuestro servicio
  • Para mejorar y personalizar la experiencia del usuario
  • Para comunicar actualizaciones y cambios
  • Para proporcionar atención al cliente
  • Para analizar patrones y tendencias de uso
  • Para enviar contenido promocional (con consentimiento)

3. Seguridad de datos

Implementamos medidas de seguridad estándar de la industria para proteger sus datos. Sin embargo, ningún método de transmisión por Internet es 100% seguro. Nos esforzamos por proteger su información personal, pero no podemos garantizar una seguridad absoluta.

4. Recopilación y uso de datos faciales

Recopilación de datos faciales

Nuestra aplicación puede recopilar imágenes faciales y datos de escaneo cuando utiliza las funciones de análisis facial. Esto incluye:

  • Imágenes faciales capturadas a través de la cámara de su dispositivo
  • Datos de características faciales para fines de análisis

Uso de datos faciales

Los datos faciales recopilados por nuestra aplicación se utilizan exclusivamente para:

  • Proporcionar la funcionalidad de análisis facial que solicita
  • Procesar sus imágenes faciales para entregar resultados de análisis

Almacenamiento y compartición

Los datos faciales se almacenan principalmente de forma local en su dispositivo. Cuando utiliza las funciones de análisis facial:

  • Los datos faciales pueden transmitirse temporalmente a OpenAI/ChatGPT solo con fines de análisis
  • No almacenamos permanentemente sus datos faciales en nuestros servidores
  • Los datos faciales no se comparten con otros terceros

Período de retención

Los datos faciales se conservan de la siguiente manera:

  • Los datos almacenados en su dispositivo permanecen hasta que desinstale la aplicación
  • Los datos transmitidos a OpenAI/ChatGPT para análisis no se almacenan permanentemente y solo se utilizan para el procesamiento inmediato de su solicitud
  • No mantenemos ningún dato facial en nuestros servidores
  • Cualquier dato que recopilemos está sujeto a un período de retención de 1 año, después del cual será eliminado

Su control

Usted mantiene el control sobre sus datos faciales mediante:

  • La posibilidad de eliminar los datos faciales almacenados localmente en cualquier momento desinstalando la aplicación

5. Cambios a esta política

Podemos actualizar esta política periódicamente. Los cambios se publicarán aquí con una fecha actualizada. Por favor, revise esta política regularmente.
